MimeTexISAA by ISAA - Italian Space and Astronanutics Association
remix by MarcoZambi, based on Mimetex MOD by Feito de Vito.

This mod requires the mimetex.cgi executable in /cgi-bin directory of
your web server.
Mimetex source code was dowloaded from here:
http://www.forkosh.com/mimetex.html
Be aware that a successful compilation of mimetex requires to satisfy
specific dependencies.
